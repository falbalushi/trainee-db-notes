# Trainee DB Notes

## Git & GitHub Basics

- **Repository**: A project folder that tracks files and changes using Git.
- **Clone**: Create a local copy of a remote repository.
- **Commit**: Save a snapshot of changes with a message.
- **Push**: Upload local commits to the remote repository.
- **Pull**: Download latest changes from the remote repository.
- **Branch**: A separate line of development.
- **Merge**: Combine changes from one branch into another.
- **Pull Request (PR)**: A request to merge one branch into another after review.

## Database Notes

- **Database**: An organized collection of structured data.
